KnightroGrade
Senior Design Project in React
KnightroGrade was my Senior Design Project at UCF, utilizing a PERN Stack with Postgres, Express, React, and Node. I primarily worked on the front-end team with React.
Code Submission & Grading System for UCF
Our project revolved around creating an automatic code grading system for UCF. Students would submit their programs to our website and have them automatically graded with test cases set by the professor, saving tedium and time for the TAs.
Front-End Manager & Figma Designer
In this project, I took on the role of front-end manager, creating the schedules and leading the design process of our website. Initial designs were crafted and refined with Figma, until we finally made them a reality with React and BootStrap.

CerealBoxd
Cross-Platform Web & Mobile App
CerealBoxd is a breakfast-themed twist on LetterBoxd, a popular movie-rating social platform. We created this social application to be cross-platform with React and React Native.
MERN Stack Project
CerealBoxd was done with a MERN Stack - MongoDB, Express, React, and Node. I was present throughout all teams on this project, primarily focusing on Mobile Front-End.
